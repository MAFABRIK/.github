# MAFABRIK IT

Repositorio central de documentación técnica, automatizaciones, consultas SQL, reportes y herramientas internas de MAFABRIK.

## Áreas principales

- **Procesos automatizados**: scripts productivos, tareas programadas, reportes automáticos y documentación operativa.
- **Consultas SQL**: consultas reutilizables, análisis de Softland y soporte a reportes.
- **Power BI**: documentación de modelos, fuentes de datos y reportes.
- **Integraciones Softland**: documentación técnica, pruebas controladas y consultas de apoyo.
- **Documentación TI**: procedimientos internos, infraestructura, servidores y estándares.
- **Herramientas internas**: utilidades de apoyo para análisis, validación y operación.

## Repositorios principales

| Repositorio | Descripción |
|---|---|
| `maf-it` | Repositorio principal del área TI. Contiene procesos, documentación y herramientas internas. |
| `procesos-automatizados` | Procesos Python, tareas programadas, archivos `.bat`, documentación y BPM. |
| `consultas-sql` | Consultas SQL organizadas por módulo, proceso o fuente de datos. |
| `powerbi-reportes` | Documentación de reportes, modelos, fuentes y métricas. |
| `integraciones-softland` | Análisis e integración controlada con Softland. |
| `documentacion-ti` | Manuales técnicos, procedimientos y diagramas internos. |
| `plantillas` | Estructuras base para nuevos procesos, README, BPM y scripts. |
